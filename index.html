1. LLMAWJ: Static Analysis Alarm Filtering Toolset Based on Large Language Model

Duration: 2023/10 - Present

Technologies: Python, LLM (GPT4, CodeLlama2), SQLite3, Static Analysis Tools (SVF, Infer)

Supervisor: Professor Xu Chang (许畅), Nanjing University, Department of Computer Science and Technology (计算机科学与技术系) SEG

Project Description: Using large language models to identify alarms generated by static analysis tools, filtering out false positive alarms, and assisting programmers in quickly addressing more suspicious alarms. Personally responsible for designing and generating the intermediate format of alarms, designing scripts for interaction with large language models, etc.

Main Responsibilities: Designed the intermediate representation format for alarms to interface with static analysis tools and large language models; designed Python scripts to convert reports generated by different static analysis tools into the intermediate format; designed Python scripts to index project source code based on the intermediate format and interact with large language models.

Project Outcomes: LLMAWJ, a static analysis alarm filtering toolset based on large language models

Project Open Source Link: https://github.com/NJU-1B348/LLMAWJ
